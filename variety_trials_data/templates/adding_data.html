Adding New Data

This is the user manual for the /add/ system.

  • {{HOME_URL}}/add/trial_entry/
    • Allows new varietal trial entries to be entered, either via a spreadsheet file or using the provided online spreadsheet editor.
  • {{HOME_URL}}/add/vareity/
    • Allows new variety entries to be entered.
  • {{HOME_URL}}/edit/variety/Variety+Name
    • Allows variety entries to be edited.
  • {{HOME_URL}}/add/location/
    • Allows new location entries to be entered.
  • {{HOME_URL}}/edit/location/Location+Name
    • Allows location entries to be edited.
